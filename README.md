# D-jame-escuchar-esa-m-sica.-
En este proyecto, analice un dataset de música para encontrar diferencias de actividad entre dos ciudades. 

Realice limpieza, correccion a valores correctos, agrupe generos y sume columnas usando  la libreria pandas de Python, 
comparé cuantas canciones se escucharon en cada ciudad para identificar patrones de comportamiento.
